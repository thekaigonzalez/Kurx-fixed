copy(2) - copy file contents to another

Author: Kai D. Gonzalez (gkai70264@gmail.com)

synopsis

            int copy (const char *_origin, const char *_dest)

description

            this function copies _origin's contents to _dest, creating a new
            file in the process (or writing over an existing one.) copy does not
            dirty any buffer - it will allocate the space needed to store and
            copy the data from _origin to _dest.
        return values & debugging
            this function does not error, however, it will give a notification
            when a file that you are trying to copy does not exist, this is to
            prevent segfaults within your programs and lua scripts.

            you do not have to worry about getting a fault with this function,
            and all memory is handled inside of it.
